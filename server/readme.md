# 📚 Student API - Express + Prisma + Jest Testing

A simple RESTful API for managing students, built with Express, Prisma (MySQL), Jest, and Supertest. Includes proper testing setup with a dedicated test database.

---

## ⚙️ Technologies Used

- **Node.js + Express**
- **Prisma ORM**
- **MySQL** (via XAMPP or other)
- **TypeScript**
- **Jest** (unit + integration tests)
- **Supertest** (for HTTP assertions)
